# Instagram Clone
Built a responsive web app clone of [Instagram](https://www.instagram.com).
The app is deployed using Vercel at : https://instagram-clone-seven-mocha.vercel.app/

## Tools and Technologies Used:
```
- React (FrontEnd)
- Firebase (Database)
- TailwindCSS (CSS)
```
