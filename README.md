# Instagram Clone
Built a responsive web app clone of [Instagram](https://www.instagram.com). <br />
The app is deployed using Vercel at : https://instagram-clone-deep512.vercel.app/
<br />
## Tools and Technologies Used:
```
- React (FrontEnd)
- Firebase (Database)
- TailwindCSS (CSS)
```
